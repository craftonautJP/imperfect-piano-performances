# Imperfect Piano Performances

A collection of piano performance data (WAV & MIDI) capturing the raw, unpolished journey of learning classical music.

## Overview
This repository archives my piano practice sessions. Unlike typical performance libraries, this collection focuses on the **process of improvement**. You will find multiple recordings of the same piece, ranging from the very first struggling attempt to a more polished version.

All pieces are classical works in the public domain.

## Key Features & Use Cases
This dataset is intentionally "imperfect" and may be useful for:
- **Creative Projects**: For those who need "realistic" or "unpolished" piano sounds rather than a perfect studio recording.
- **Machine Learning**: A unique source for training models on human learning patterns, error detection, or style transition.
- **Musical Analysis**: Analyzing how fingering, dynamics, and temporal expression evolve as a player gains proficiency.

## Data Specifications
- **Instrument**: Roland DP603 (Digital Piano)
- **Formats**: 
  - `WAV`: High-quality audio captured directly from the output.
  - `MIDI`: Precise event data including note velocity and pedal movements.
